# OOP Exercise Collection

This repository contains various Object-Oriented Programming exercises in Java, cleaned and optimized for terminal usage.

## Repository Structure

Each folder represents an exercise:
- `EscSimilEsonero24`: Device management simulation (Smartphone/Tablet).
- `EsoneroSimulation_24`: Museum paintings management (Figurative/Abstract).
- `EsoneroSimulation25`: Museum dinosaurs management (Herbivore/Carnivore).

## How to Compile and Run

To compile and run an exercise from the terminal, navigate into its directory and use `javac` and `java`.

### Example: EsoneroSimulation25

1. **Navigate to the directory:**
   ```bash
   cd EsoneroSimulation25
   ```

2. **Compile all Java files:**
   ```bash
   javac *.java
   ```

3. **Run the main class:**
   ```bash
   java Museo
   ```

### Main Classes for each exercise:
- **EscSimilEsonero24**: `Negozio`
- **EsoneroSimulation_24**: `Museo`
- **EsoneroSimulation25**: `Museo`

## Cleaning up
To remove the compiled `.class` files, you can run:
```bash
rm *.class
```
